清除Teamspeak3服务器黑名单，连接到低版本服务器

发表于 2019-09-28 更新于 2020-02-07 分类于 Tools 阅读次数：

为什么会有这个东西

由于某原因仍有很多人正在使用某盗版服务器提供商的服务器

因此需要使用“中文破解版”连接至他们的服务端，但当你需要同时连接新旧版服务器时，你会发现旧版服务器已被列入黑名单，所以你需要清除黑名单

在使用这个批处理前，请先考虑使用正版服务器（免费32人槽位）

怎么使用

新建一个文本文档，后缀改为.bat，复制粘贴保存，管理员运行

或者直接下载本文件，然后管理员运行
