轻松搭建Teamspeak3服务器

发表于 2019-09-24 更新于 2020-01-25 分类于 Tutorial 阅读次数：

为什么要使用Teamspeak3作为语音服务软件

简单易用，无广告，内存占用小当你电脑⑧太行的时候，内置文件服务器，有丰富的快捷键以及自定义选项，无许可证即可享有超大32人服务器
某10槽位盗版服务器一个月要多少钱来着？

那么在哪里能获取到适合作为Teamspeak3服务器的vps呢

国内各大云服务器学生机，超级便宜，例如阿里云，腾讯云等
BTW一个vps除了做teamspeak服务器还能做很多事

在本文编写时最新服务端版本为3.10.2

然后你会看到这么一大段东西

其中 Server Query Admin Account 我们暂且不管，这是Teamspeak3的query功能，大概会在另一篇文章里简单介绍

重要的是 ServerAdmin privilege key ，这是你第一次登陆服务器用于获取管理员权限的密匙，管理员可以在客户端对服务器进行一定的设置

Teamspeak服务端端口

一般情况下，Teamspeak服务器会使用以下几个端口

  • 9987/udp 语音服务器端口
  • 30033/tcp 文件传输端口
  • 10011/tcp ServerQuery端口

请开放你的vps对应的端口以及对应的协议，一般可以在服务商面板的防火墙/网络规则中找到

同时，这些端口都是可以更改的，此处不再赘述，请查阅Reference
