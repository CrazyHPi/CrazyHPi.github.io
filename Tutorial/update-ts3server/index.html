Teamspeak服务器更新教程

发表于 2020-02-06 更新于 2020-02-07 分类于 Tutorial 阅读次数：

写在前面

为什么会有这个东西呢，最近有几个旧版本的ts服务器突然连接不能，服务端重启提示 Failed loading default license file! ，就寻思着把服务器更新一下好了

Let’s get started

首先停止服务器

如果你是跟着我的安装教程装的服务器，可以使用 service ts stop 来停止服务器，或者进入ts服务端文件目录，使用 ./ts3server_startscript.sh stop

备份数据库

如果你是使用mysql或者mariadb请自行备份对应的数据库

如果你使用的是自带的sqlite，用你喜欢的方式备份数据库文件 ts3server.sqlitedb

下载新的服务器文件

现在最新版为3.11.0，请在Teamspeak官网获取最新服务器文件

对没错，就是直接覆盖过去就完事了

再启动服务器

进入目录 ./ts3server_startscript.sh start 或者 service ts start 就完事了

再客户端登陆服务器，虽然不是同一个版本，但还是同一个原汁原味的服务器，奥利给

完全不需要rm重新来过，再启动发现没原来内味了

Reference
